# Sistema de Inventario (Java + SQL)

## 📌 Descripción
Aplicación de escritorio desarrollada en Java para la gestión de inventarios. 
Permite administrar productos, usuarios y operaciones mediante un sistema completo con autenticación, reportes y visualización de datos.

---

## ⚙️ Funcionalidades

- 🔐 Login con usuario y contraseña
- 🔄 Recordatorio para cambio de contraseña (cada 30 días)
- 📦 Gestión de inventario (CRUD)
- 👥 Gestión de usuarios
- 📊 Generación de gráficos
- 🧾 Generación de reportes con iReport
- 💾 Respaldo de base de datos
- 📈 Conteo y control de inventario

---

## 🛠️ Tecnologías utilizadas

- Java
- MySQL (o SQL que uses)
- Swing (interfaz gráfica)
- iReport (reportes)
